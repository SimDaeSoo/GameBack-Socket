Game Project / Back End (Socket Server)
=============

### How To Run? ###
- dependency install
```sh
$ npm install
```
- build and run as develop mode
```sh
$ npm start
```
- build and run as production mode
```sh
$ npm run build
```

### Server ###
- AWS EC2 Instance
- PM2
- Node.js
- Express
- Socket.io

### Front Dev Environments ###
- Typescript
- Javascript
- Webpack 4
